# Almost Rock Paper Scissors

A two button version of rock paper scissors.

## How to play
```
ruby rps.rb
```

## Sample Output
```
---------------------------
--  Rock Paper Scissors? --
---------------------------

Select an option:

1: Rock
2: Paper
> 2
Paper vs Paper
Tie

---------------------------
--  Rock Paper Scissors? --
---------------------------

Select an option:

1: Scissors
2: Paper
> 1
Scissors vs Paper
Player Wins

---------------------------
--  Rock Paper Scissors? --
---------------------------

Select an option:

1: Rock
2: Scissors
> 1
Rock vs Rock
Tie
```
